# What `Redux` Does?

Easily manage global #states on React with #reducer like but in a simpler way. Will replace #context #API.


## How to Use

1. Install `Redux Toolkit` and `React bindings`:

```sh
npm install "@reduxjs/toolkit" "react-redux"
```

2. Create a `stores/` under `src/`.
3. 4. Create a JavaScript file meaningfully named file for your state.
4. Write these:

```jsx
import { createSlice } from "@reduxjs/toolkit"

const myState = createSlice({
  name:"myState",
  initialState: 0,
  reducers:{
	myAction: (state, action) => {
	  state = action.payload
	}
  }
})

export const { myAction } = myState.actions
export default myState.reducer
```

5. Create an `index.js` under `stores/` to configure stores:

```jsx
import { configureStore } from "@reduxjs/toolkit"
import myState from "./myState

const store = configureStore({
  reducer: {
    myState
  }
})

export default store
```

6. Use `Provider` in `index.js`:

```jsx
import { Provider } from "react-redux"
import store from "./stores"

root.render(
  <Provider store={store}>
	<App />
  </Provider>
)
```

7. To access methods, use `useDispatch` #hook:

```jsx
import { useDispatch } from "react-redux"
import { myAction } from "../stores/myState"

const Component = () => {
	const dispatch=useDispatch()
	... {
	  dispatch(myAction())
	}
  ...
}
```

8. To access states, use `useSelector` #hook:

```jsx
import { useSelector } from "react-redux"

const Component = () => {
  const { myState } = useSelector(state => state.myState)
  ...
}
```

## Optional Usages

Can create a `utils.js` under `src/` to create custom handlers:

```jsx
import store from "./stores"
import { myAction } from "./stores/myState"

export const myStateHandler = state => {
	  store.dispatch(myAction(state))
}
```

And just use this below to access state setters:

```jsx
import { myStateHandler } from "../utils"

export default Component = () => {
  myStateHandler(1)
}
```
